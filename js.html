Quelques notions acquises

Utilisation du DOM, Document Object Model

Dans le cadre de mes projets web, j'ai appris à manipuler le DOM (Document Object Model) pour rendre mes pages interactives.
J’ai utilisé JavaScript pour modifier dynamiquement le contenu HTML, changer des styles CSS, et réagir à des événements comme les clics ou les entrées clavier.
Par exemple, j’ai pu développé un formulaire interactif vérifiant les données saisies en temps réel et affichant des messages d’erreur sans recharger la page.
Cette compétence m’a permis d'améliorer l’expérience utilisateur et de rendre mes interfaces plus dynamiques.




Utilisation des appels AJAX avec l'utilisation de la fonction fetch()

J’ai appris à utiliser la fonction fetch() en JavaScript pour effectuer des requêtes HTTP asynchrones.
Cela m’a permis de récupérer des données depuis une API (comme une API météo ou une base de données interne) et de les afficher dynamiquement sur une page web. J’ai ainsi pu créer une interface utilisateur qui se met à jour automatiquement sans rechargement complet, en utilisant les formats JSON et les promesses pour traiter les réponses serveur.
Cette compétence est essentielle dans le développement d’applications web modernes.
